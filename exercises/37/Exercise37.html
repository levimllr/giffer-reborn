Exercise 37: Create a function named sequenceLEDsInAZigZagMovingUp that uses three parameters to control delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function.

 

ACTIVITIES:

 

  1. When you finish this exercise there will be one custom function in loop(), sequenceLEDsInAZigZagMovingUp.
  2. Copy the program from Starting Point Exercise37 into the Arduino IDE
  3. Create  a new function named sequenceLEDsCounterClockwise that uses parameters to control the delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function. The leds should blink in a counter clockwise rectangle starting at LED 2 - LED 8 then LED 15 - LED 9
  4. Create  a new function named  sequenceLEDsInAZigZagMovingUp that uses three parameters to control the delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function. The leds should blink in a zig zag counter clockwise rectangle starting at LED 2, LED 9, LED 3, LED 10, LED 4, LED 11, LED 5, LED 12, LED 6, LED 13, LED 7, LED 14, LED 8, LED 15
  5. You should be able to look at the image on giffer to determine the correct onTime and offTime and repeatCount
  6. loop() repeats 3 times, then there should be a delay of 3 seconds after which the whole thing starts over again.
  7. Save the completed sketch as YourInitials_Exercise37 in: Documents – Arduino – Exercises
  8. Copy the code into the Giffer website
  9. Enter your name and exercise number.
  10. Select the Run and Grade button.
  11. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  12. If it is Correct - use the Copy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
