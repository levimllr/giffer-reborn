Exercise11: Using a "while" loop to blink each LED once.

 

NEW:  Incrementing by more than one x = x +2;

 

DESCRIPTION: x gets the value of x + 2. Allows us to count up by two.

 

Other valid options include:

                        x = x + 5; // x gets the value of x + 5

                        and so forth

ACTIVITIES:

  1. Copy the program from Starting Point Exercise11 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED’s 2 through 15 blink on and off in order with a 200ms delay between each, then the program pauses for 4 seconds and the whole thing starts over again…….
  3. Modify the program so that only LED’s 2-8 blink on and off..
  4. Modify the program so that LED’s 9-15 blink on and off.
  5. Modify the program so that only LED’s 2, 4, 6, 8, 10, 12, 14 blink on and off.
  6. Save the completed sketch as YourInitials_Exercise11 in: Documents – Arduino – Exercises
  7. Copy the code into the Giffer website
  8. Enter your name and exercise number.
  9. Select the Run and Grade button.
  10. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  11. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  12. Go to CL 10.3.11 Upload Completed Exercise11:  And upload evidence that you have completed Exercise11
  13. Click "Prepare an answer" on NEO
  14. Click the "<>" (Source code) button.
  15. Paste by pressing CTRL + V
  16. Click "Save and submit for grading"
  17. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise11 - in your Documents/Arduino/Exercises Folder.
