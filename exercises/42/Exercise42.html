NEW: Arrays in the Arduino programming environment; 

DESCRIPTION: An array is a collection of variables that are accessed / referenced with an index number. Essentially an array is just a way to store a list on numbers. Because we can put values in this list in any order, using arrays to program a light sculpture becomes really useful.

 

Some vocabulary:

Array: A list of things. For now it will typically be a list of integers

Elements: The actual values stored in the list

Number of elements: count up how many integers in the list

Index: A way to reference an element in the array (a way to list a number in the list)

 

EXAMPLE:

 

// Creates an array of 5 integers / elements and assigns values to those integers

 

int myArray[5] = {4, 7, 2, 8, 3};    // {4, 7, 2, 8, 3 } <- array values

 

Arrays are zero-indexed, so the first element of the array is at index 0 not index 1

myArray;

// { 4, 7, 2, 8, 3 } <- array values

// [ 0  1 2 3 4 ] <- index number

 

To access a element in an array - Do the following

myArray[0] = 4;   // This statement allows us to access the first element of the array

// { 4, 7, 2, 8, 3 } <- array values

// [ 0  1 2 3 4 ] <- index number

 

myArray[4] = 3;  // This statement allows us to access the fourth element of the array

// { 4, 7, 2, 8, 3 } <- array values

// [ 0  1 2 3 4 ] <- index number

 

EXAMPLE:

 

void loop()

{

 int myFirstArray[7] = {

   2, 4, 6, 8, 10, 12, 14  }; // creates an array of 7 integers / elements and initializes / assigns values to those integers  

 int index;

 // Blinks LED's in the order defined by the array

 for(index = 0; index <= 6; index++)

 {

   digitalWrite(myFirstArray[index], HIGH);  

   delay(150);

   digitalWrite(myFirstArray[index], LOW);

 }

 delay(4000);

}
