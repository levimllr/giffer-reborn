Exercise 50: Using arrays in custom functions, to turn on and off LEDs in very unique patterns that would otherwise be hard to program in a loop.

 

ACTIVITIES:

  1. Start with your completed code from exercise49 and save it as exercise50 as a starting point for this exercise.
  2. Change the name of the function to ArraysSequenceShowAllTheColors that uses three parameters to control delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function.
  3. In the function, re-name the array LEDsCounterClockwiseStartAt15, and modify the number of elements to fit the requirements of the program.
  4. Modify the function to use the array to get LED’s blink on and off in the following sequence: 15, 14, and so on down to 9, then 2, and so on up to 8.  
  5. You should be able to look at the image on giffer to determine the correct onTime and offTime and repeatCount(3)
