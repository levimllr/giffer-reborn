Exercise13: Counting backwards and forwards using a "while" loop.

 

NEW:  Two while loops one counting up then another counting down;

 

                        Use one while loop to sequence LEDs 2 through 8.

                        Then use a second while loop to sequence LEDs 7 through 3.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise13 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED’s 2 through 15 blink on and off in order with a 200ms delay between each, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Modify the program so that only LED’s 2 through 8 blink on and off, then LED’s 7, 6, 5, 4, 3 blink on and off.
  4. Save the completed sketch as YourInitials_Exercise13 in: Documents – Arduino – Exercises
  5. Copy the code into the Giffer website
  6. Enter your name and exercise number.
  7. Select the Run and Grade button.
  8. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  9. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  10. Go to CL 10.3.13 Upload Completed Exercise13:  And upload evidence that you have completed Exercise13
  11. Click "Prepare an answer" on NEO
  12. Click the "<>" (Source code) button.
  13. Paste by pressing CTRL + V
  14. Click "Save and submit for grading"
  15. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise13 - in your Documents/Arduino/Exercises Folder.
