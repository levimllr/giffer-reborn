NEW: Using parameters in our function definitions;

 

EXAMPLE:

 

void blink2Once(int t)
{
 digitalWrite(2, HIGH);
  delay(t);
  digitalWrite(2, LOW);
  delay(t);
}

 

In this example the function blink2Once is expecting an argument (in this case an integer) to be passed to it when you call it. If we look at this code it, we can see that It is using this argument to determine the delay time.
