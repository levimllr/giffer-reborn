Exercise08: Count in a loop to blink 4 times.

Make sure you have done the following before proceeding:

  1. Have watched video on while loops /files/1883896/Arduino_Tutorial_3_-_While_Loops(2).mp4
  2. Have looked at and understand while loop handout DPEA_Looping_With_While.pdf

 

 

NEW:  Multiple variables waitTime and count;

 

DESCRIPTION: using multiple variables to control our program.

 

NEW:  while loop;

 

DESCRIPTION: A way to repeat commands until a condition is no longer true.

 

Example: The while loop will execute the code between the while loop parenthesis while the condition is true, then the while loop will end.

 

NEW:  count = count + 1;

 

DESCRIPTION: This is NOT algebra – rather a short hand notation of:

 

“count gets the value of count + 1”
