Exercise03: Turn on three LEDs, delaying 1 second between each one.

 

NEW:  delay (1000);

DESCRIPTION: Tells Arduino to wait 1000ms or 1 second before going to the next line.

 

GENERAL USAGE: delay (X);

DESCRIPTION: Tells Arduino to wait X milliseconds before going to the next line.

 

ACTIVITIES:

  1. Copy the program from Starting Point Exercise03 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify thatLED’s 2, 3, and 4  light up with a one second delay between each.
  3. Modify the program so the Arduino only delays 1/2 second between each LED.
  4. Modify the program so the Arduino only delays 1/4 second between each LED.
  5. Modify the order the LEDs go on. New order LED 4, then LED 3, then LED 2.
  6. Save the completed sketch as YourInitials_Exercise03 in: Documents – Arduino – Exercises
  7. Copy the code into the Giffer website
  8. Enter your name and exercise number.
  9. Select the Run and Grade button.
  10. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  11. If it is Correct -  use the Copy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  12. Go to CL 10.3.02 Upload Completed Exercise03:  And upload evidence that you have completed Exercise03
  13. Click "Prepare an answer" on NEO
  14. Click the "<>" (Source code) button.
  15. Paste by pressing CTRL + V
  16. Click "Save and submit for grading"
  17. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise03 - in your Documents/Arduino/Exercises Folder.
