Exercise 45: Using arrays in custom functions, to turn on and off LEDs in very unique patterns that would otherwise be hard to program in a loop.

You MUST create descriptive and appropriate variable names. Save yourself some time and copy the required Array names and required function names from this document and paste them into the sketch. Hints are suggestions – not requirements.

NEW: You can use the following methods to access the values stored in an array.

 

int getArrayValue;

 

int myArray[5] = {10, 7, 13, 9, 8};

 

getArrayValue = myArray[1];  // Puts 7 in getArrayValue;

 

getArrayValue = myArray[0];  // Puts 10 in getArrayValue;

 

myArray[0] = 6;   // Puts a 6 in the first element
// { 6, 7, 13, 9, 8 }

myArray[4] = 0;   // Puts a 0 in the fifth / last element
// { 6, 7, 13, 9, 0}

myArray[1] = myArray[3]; // Puts a 9 in the second element
// { 6, 9, 13, 9, 0}

myArray[3] = 15; // Puts a 15 in the fourth element
// { 6, 9, 13, 9, 15}

You can also use the following embedded expressions to access / store values in an array:

myArray[1+1] = 2; // Puts a 2 in myArray at element 3

// { 6, 9, 2, 9, 15}

int lastElementIndex = 4;

myArray[lastElementIndex] = 1; // Puts a 1 in myArray at element 5

// { 6, 9, 2, 9, 1 }

myArray[lastElementIndex-1] = 0; // Puts a 0 in myArray at element 4

// { 6, 9, 2, 0, 1 }
