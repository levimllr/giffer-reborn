Exercise18: Two for loops in a row.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise18 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED 2 blinks on and off four times with a 200ms delay between each, then LED 3 blinks on and off four times with a 200ms delay between each, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Modify the program so that it blinks LED 2 four times, then it blinks LED 3 four times, then it blinks LED 4 four times waits four seconds THEN - all LEDS are off for four seconds then everything starts over again.
  4. Modify the program so that it blinks LED 2 two times, then blinks LED 3 three times, then blinks LED 4 four times THEN - all LEDS are off for four seconds then everything starts over again.
  5. Save the completed sketch as YourInitials_Exercise18 in: Documents – Arduino – Exercises
  6. Copy the code into the Giffer website
  7. Enter your name and exercise number.
  8. Select the Run and Grade button.
  9. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  10. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  11. Go to CL 10.3.18 Upload Completed Exercise18:  And upload evidence that you have completed Exercise18
  12. Click "Prepare an answer" on NEO
  13. Click the "<>" (Source code) button.
  14. Paste by pressing CTRL + V
  15. Click "Save and submit for grading"
  16. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise18 - in your Documents/Arduino/Exercises Folder.
