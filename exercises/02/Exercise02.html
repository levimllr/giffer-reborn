Exercise02: Turn on an LED.

 

NEW:  digitalWrite(2,HIGH);

DESCRIPTION: Tells Arduino to send enough power to make LED attached to pin 2 light up.

 

ACTIVITIES:

  1. Copy the program from Starting Point Exercise02 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED 2 lights up.
  3. Exercise: Remove the; from the end of the line: digitalWrite(2, HIGH);
  4. Try to compile and upload the program. You should get a compiler error.
  5. Look at the compiler error (red cryptic message at bottom of screen) – this it what happens when we are missing a semi-colon
  6. Fix the program so it compiles (put the ; back in at the end of digitalWrite(2, HIGH);
  7. Verify that there is no compiler error and that LED 2 lights up.
  8. Modify the program so the Arduino Turns on LEDs 2 and 3.
  9. Modify the program so the Arduino Turns on LEDs 2, 3 and 4.
  10. Save the completed sketch as YourInitials_Exercise02 in: Documents – Arduino – Exercises
  11. Copy the code into the Giffer website
  12. Enter your name and exercise number.
  13. Select the Run and Grade button.
  14. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  15. If it is Correct -  use the Copy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  16. Go to CL 10.3.02 Upload Completed Exercise02:  And upload evidence that you have completed Exercise01
  17. Click "Prepare an answer" on NEO
  18. Click the "<>" (Source code) button.
  19. Paste by pressing CTRL + V
  20. Submit
  21. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise02 - in your Documents/Arduino/Exercises Folder.
