Exercise 55: Continuing to Learn how to use the Knob / Potentiometer on the Light Sculpture board

 

NEW: Reading the rotation state of the knob / potentiometer on the Light Sculpture board

 

DESCRIPTION:

 

Last exercise we learned that the potentiometer has a on / off switch. The potentiometer also has a variable resistor that the Arduino interprets as a value from 0 to 1023. This variable resistor is connected to pin A5 on the Arduino. Note this is pin A5 versus pin 5. The A in A5 denotes that it is an analog pin which means that instead of values 0 (LOW) or 1 (HIGH) it has 1024 values between 0(LOW) and 1023 (HIGH).

 

Because we can in theory distinguish 1024 different values we can tell the Arduino to do many different things based on what value we read from A5.

 

So now we know there are two pins connected to the potentiometer. 18 and A5. 18 is a digital pin - with valid read values 0 (LOW) and 1 (HIGH). A5 is analog with valid read values 0 (LOW) all the way up to 1023 (HIGH).  

 

Pin 18 on the Arduino is connected to the potentiometer like an ON / OFF switch. If you rotate the Knob from all the way clockwise to all the way counterclockwise you will hear a click at the end of its counterclockwise rotation. This click is connecting Pin 18 on the Arduino to ground. If we rotate the Knob clockwise you will the click again and that means that Pin 18 is no longer connected to ground. At the point of clockwise rotation just after you hear the click which indicates pin 18 is connected to 5V (or is HIGH) pin A5 should read 0 (LOW). As you continue to rotate clockwise A5 will increase up to a read value of 1023 (HIGH).

 

To read the analog value on A5, we need to do the following:

 

int value = analogRead(5);

 

analogRead() is a function that can read analog values from pins on the Arduino. Just like pinMode() or digitalWrite(), analogRead() needs a pin to use.

 

int value = analogRead(5); // This line sets the variable “value” equal to the state pin A5 is in.



EXAMPLE:

 

void setup()

{

 Serial.begin(9600); // Setup the serial port speed

}

 

void loop()

{

 int pinA5 = 0;     // variable to store the value of pin A5

 pinA5 = analogRead(5); // read the input pin valid values 0-1023

 

 if (pinA5 < 512)

 {

  Serial.print("Pin A5 is ");

   Serial.println(pinA5);

   Serial.println("Pin A5 is less than 512");

 }

 

 if (pinA5 > 511)

 {

   Serial.print("Pin A5 is ");

   Serial.println(pinA5);

   Serial.println("Pin A5 is more than 511");

 }

 delay(500);

}

 

If we run this code on a Arduino with a light sculpture connected and look at the serial output we will see the output change based on the position of the knob.
