Exercise 39: Create a function named sequenceLEDsRedsThenGreens that uses three parameters to control delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function.

 

ACTIVITIES:

 

  1. When you finish this exercise there will be one custom function in loop(), sequenceLEDsRedsThenGreens.
  2. Create  a new function named sequenceLEDsRedsThenGreens that uses three parameters to control the delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function. The leds should blink Reds (2,4,6,8) on then off then Greens (3,5,7) on and off.
  3. You should be able to look at the image on giffer to determine the correct onTime and offTime and repeatCount
  4. loop() happens once, then there will be a 3 second delay after which the whole thing starts over again.
  5. Save the completed sketch as YourInitials_Exercise39 in: Documents – Arduino – Exercises
  6. Copy the code into the Giffer website
  7. Enter your name and exercise number.
  8. Select the Run and Grade button.
  9. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  10. If it is Correct - use the Copy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  11. Go to CL 10.4.09 Upload Completed Exercise39: And upload evidence that you have completed Exercise39
  12. Click "Prepare an answer" on NEO
  13. Click the "<>" (Source code) button.
  14. Paste by pressing CTRL + V
  15. Click "Save and submit for grading"
