Exercise 44: Using arrays in custom functions, to turn on and off LEDs in very unique patterns that would otherwise be hard to program in a loop.

 

You MUST create descriptive and appropriate variable names. Save yourself some time and copy the required Array names and required function names from this document and paste them into the sketch. Hints are suggestions – not requirements.

ACTIVITIES:

 

  1. Copy the program from Starting Point Exercise44 into the Arduino IDE
  2. Compile and upload this program and verify that all the even LED’s blink on and off with a 150ms delay between each, then the program pauses for 4 seconds and the whole thing starts over again….
  3. Create a new function named ArraysSequenceLEDsCounterClockwiseStartAt2 that uses three parameters to control delay onTime, the delay offTime and repeatCount the number of times a for loop repeats in the function.
  4. In the new function, create an array with 14 elements named LEDsCounterClockwise
  5. Use a for loop in the new function that blinks on / off  LED’s 2,3,4 up to 8 then LED’s 15, 14, 13 down to 9. The for loop must use the values in the array LEDsCounterClockwise
  6. Use 100ms for onTime and offTime, and use 3 as repeatCount,
  7. loop() happens once, then there will be no delay before the whole thing starts over again.
  8. Save the completed sketch as YourInitials_Exercise44 in: Documents – Arduino – Exercises
  9. Copy the code into the Giffer website
  10. Enter your name and exercise number.
  11. Select the Run and Grade button.
  12. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  13. If it is Correct - use the Copy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  14. Go to CL 10.4.14 Upload Completed Exercise44: And upload evidence that you have completed Exercise44
  15. Click "Prepare an answer" on NEO
