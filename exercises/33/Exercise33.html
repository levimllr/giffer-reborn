Exercise 33: Making your functions more powerful by adding multiple parameters, to make repetitive tasks easier.


Make sure you have done the following before proceeding:

 

  1. Have watched the fourth video on functions - Arduino Functions 4

 

NEW: Using two arguments in our function definitions;

 

EXAMPLE:

 

void blinkOnce(int t, int n)
{
 digitalWrite(n, HIGH);
  delay(t);
  digitalWrite(n, LOW);
  delay(t);
}

 

In this example the function blinkOnce is expecting two arguments. In this case it is expecting an integer t, that will be used to determine the delay time, and an integer n which will be used to determine the LED to blink on and off.
