Exercise 35: Using a parameter to control the number of times a for loop repeats in a function.

 

NEW: Using a parameter to control the number of times a for loop repeats in a function.

 

EXAMPLE: In the function definition sequenceLEDsFromRightToLeft below - you can see that the integer onTime is used to control the delay time and the integer repeatCount is used to control the number of times a for loop repeats in the function sequenceLEDsFromRightToLeft.

 

void sequenceLEDsFromRightToLeft(int onTime, int repeatCount)
{
 int count;
 int pinNumber;
 
 for(count = 1; count <= repeatCount; count++)
 {
   for(pinNumber = 2; pinNumber <= 15; pinNumber++)
   {
     digitalWrite(pinNumber, HIGH);
     delay(onTime);
     digitalWrite(pinNumber, LOW);
   }
 }
}
