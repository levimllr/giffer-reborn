Exercise 36: Using two parameters to control onTime and offTime in a function.

 

NEW: Using a two parameters to control onTime and offTime in a function

 

EXAMPLE: In the function definition sequenceLEDs2Through8 below - you can see that the integer onTime is used to control the delay time that the LED is on, and offTime is used to control the delay time that the LED is off, and the integer repeatCount is used to control the number of times a for loop repeats in the function sequenceLEDs2Through8.

 

void sequenceLEDs2Through8(int onTime, int offTime, int repeatCount)
{
 int count;
 int pinNumber;
 
 for(count = 1; count <= repeatCount; count++)
 {
   for(pinNumber = 2; pinNumber <= 8; pinNumber++)
   {
     digitalWrite(pinNumber, HIGH);
     delay(onTime);
     digitalWrite(pinNumber, LOW);

     delay(offTime);
   }
 }
}
