NEW: function definitions;

DESCRIPTION: Creating your own function is a easier way to repeat a set of commands.

EXAMPLE:

 

void blink()    // this is called a function declaration
{
   digitalWrite(2, HIGH);       // This is the body of the function
    delay(500);
   digitalWrite(2, LOW);
   delay(500);
 }

 

Creating Functions

When you create a function: you have to do the following:

(1) declare the function

(2) specify what arguments if any, the function “accepts”

(3) write the function’s code in the body.

So a function has three main components: declaration, arguments, and body

Declaration: the declaration of a function tells the machine two specific things: return type and function name. You must always have both a return type and a function name for every function declaration.

 

return type – what is the type of information this function will return when it gets called?

void – returns nothing

int – returns an integer ( To Start - our functions will not return a values.)

 

function name – what is the unique name this function has? This name will be used every time the function needs to get called

 

Example function declaration:

   void  blinkOnce

(return type)  (function name)

 

Arguments

You can pass values to a function when you call it. An argument is something that describes what types of values a function will accept. For example:

int blinkOnce(int ledNumber, int delay);

 

Accepts two arguments. The first argument must be an integer, the second argument must also be an integer. The function must be called with two values:

 

blinkOnce(4, 200);

 

When a function gets called, it will set variables to the values that get passed to it when it is called. So from the example above, when you call  blinkOnce(4,200) then  ledNumber gets set to 4, delay gets set to 200 and then the body of the function runs once.

 

Body

The body of the function is the actual code that gets to run when a function is called. A matching pair of opening and closing braces specifies the start and end of a function’s body. For example:

int doNothing()

    {

// body code goes here

}

 

The body code works the same as all other code in your program: you can create variables, assign values to variables, do arithmetic, run loops, have if/else statements, and call other functions. The only thing you cannot do inside the body of a function is declare other functions.
