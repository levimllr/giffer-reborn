Exercise 31: Creating our own functions, to make repetitive tasks easier. Up till now, we have been using built in functions such as digitalWrite and delay. Now we are going to learn to create our own functions to make repetitive tasks easier.

Make sure you have done the following before proceeding:

  1. Have watched the first video on functions mp4
  2. Have looked at and understand the Functions hand out  DPEA_Looping_With_For.pdf

 

NEW:  function defintions;

 

DESCRIPTION: A easier way to repeat a set of commands.

 

NEW:  X++;

 

DESCRIPTION: A quicker and easier way to increment a variable.

 

Example:      X++;   is short hand for X = X +1;

 

NEW:  X--;

 

DESCRIPTION: A quicker and easier way to decrement a variable.

 

Example:      X--;   is short hand for X = X -1;

ACTIVITIES:

  1. Start with a blank sketch in the Arduino IDE and create a for loop in setup that tells all the pins connected to LEDs 2-15 to be outputs.
  2. Next create a for loop in loop that repeats fifteen times.
  3. Next create a list of digitalWrites that blinks on and off the LED’s in the following order: 15, 8, 7, 14, 13, 6, 5, 12, 11, 4, 3, 10, 9, 2. This order is too unusual to use a nested for loop. We will learn how to do this more easily later with Array Strings.
  4. Delay four seconds and then start at the beginning again.
  5. Make Sure you are using variables in your delays. For example: delay(waitTime); instead of delay(100);.
