Exercise 56: Continuing to Learn how to use the Knob / Potentiometer on the Light Sculpture board

 

NEW: Reading both the ON / OFF state and the rotation state of the knob / potentiometer on the Light Sculpture board. This is very similar to the final program you will be running on your light sculpture.

 

DESCRIPTION:

 

In the previous exercises we have learned about how the potentiometer affects two pins on the Arduino. We have used the functionality of each pin separately in a program. In this exercise we are going to have our program use the functionality of both pins.

 

Specifically we are going to have the program look at the state of pin18 (the on / off switch) and the position of the potentiometer to see the value of the A5 pin.
