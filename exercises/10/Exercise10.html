Exercise10: Turn on all the LED's - one at a time.

ACTIVITIES:

  1. Copy the program from Starting Point Exercise10 into the Arduino IDE.
  2. Compile and upload this program to the Arduino and verify that LED’s 2 through 15 light up in order with a 500ms delay between each.
  3. Create a while loop in setup() to configure all the pins as outputs. Hint: create a variable in setup() named ledNumber..
  4. Modify the program so the delay between turning on the LED’s uses a variable.
  5. Save the completed sketch as YourInitials_Exercise10 in: Documents – Arduino – Exercises
  6. Copy the code into the Giffer website
  7. Enter your name and exercise number.
  8. Select the Run and Grade button.
  9. Look at the resulting Output image and make sure it is correct (Correct at lower part of image)
  10. If it is Correct -  use theCopy Page button to copy the evidence that you have correctly completed the exercise to clipboard.
  11. Go to CL 10.3.10 Upload Completed Exercise10:  And upload evidence that you have completed Exercise10
  12. Click "Prepare an answer" on NEO
  13. Click the "<>" (Source code) button.
  14. Paste by pressing CTRL + V
  15. Click "Save and submit for grading"
  16. If you made changes to the code in Giffer, then Copy the code from Giffer and paste it into the Arduino and re-save it as: YourInitials_Exercise10 - in your Documents/Arduino/Exercises Folder.
